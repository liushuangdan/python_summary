Matplotlib常见设置和操作

import matplotlib.pyplot as plt

plt.style.use('seaborn')
plt.rcParams['font.family'] = ['Arial Unicode MS', 'Microsoft Yahei', 'SimHei', 'sans-serif']

Jupyter notebook 图像输出的三种方式

  1. 网页内显示图像 (默认）

  2. GUI软件界面显示图像，可操作性强

  3. 在网页内显示GUI版图像，结合了前两种的好处

切换输出的时候一般需要重启内核

网页内显示

# 默认网页内显示

%matplotlib inline

plt.title("消费水平")
plt.plot([2,3,4,5,8,9,10])
