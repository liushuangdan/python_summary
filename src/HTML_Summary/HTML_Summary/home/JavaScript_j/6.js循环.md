js循环
===

---

> 程序中进行有规律的重复性操作，需要用到循环语句。
>
> break 和 continue 语句对循环中的代码执行提供了更严格的控制。


* ### for循环

    ```js
    for(var i=0;i<len;i++){
        ......
    }
    ```

* ### while循环

    ```js
    var i=0;

    while(i<8){

        ......

        i++;
    }
    ```

* ### for-in 语句

    > for-in 语句是严格的迭代语句，用于枚举对象的属性。

    ```js
    var a = [10,20,30,40,50];
    //迭代的是数组的下标。
    for(i in a){
    document.write(a[i]);
    }
    //输出： 1020304050
    ```
