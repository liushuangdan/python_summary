js流程控制
===

---

> 流程控制用于基于不同的条件来执行不同的动作。

* ### If 语句

    > if... else ...
    >
    > if ... else if ... else...
    > 
    > 可以单分支,双分支,也可以多分支,需要注意 else if中间必须要有空格

    ```js
    if (condition){
    //当条件为 true 时执行的代码
    }else{
        //当条件不为 true 时执行的代码
    }
    ```

* ### switch 语句

    > 多分支语句： switch（）{case ：......}
    >
    > switch 语句用于基于不同的条件来执行不同的动作。

    ```js
    switch(n){
        case 1:
            //执行代码块 1
            break;
        case 2:
            //执行代码块 2
            break;
        default:
            //与 case 1 和 case 2 不同时执行的代码
    }
    ```
