Introduction

js运算符

算 字 赋 比 逻 位 它

  • 算术运算符

      +       加
  -       减 
  *       乘
  /       除
  ++      自增1
  --      自减1

  • 字符串连接

      +       两个字符串相加

  • 赋值运算

      =       将一个值赋给另一个
  +=      将一个值加掉等号右边的数后在赋值给左边
  -=      将一个值减掉等号右边的数后在赋值给左边
  %=      将一个值除以等号右边的数后在赋值给左边

  • 比较运算符

      <       小于
  >       大于
  >=      大于等于
  <=      小于等于
  ==      等于等于，可以忽略数值类型，比如'1' == 1
  ===     全等于，连数据类型也得相等
  !=      不能于，可以忽略数值类型
  !==     全不等于，连数值类型

  • 逻辑运算符

      &&      逻辑与
  ||      逻辑或
  !       逻辑非

  • 位运算

      ^       异或
  &       and
  |       or
  <<      右移
  >>      左移
  ~       按位取反

  • 其它运算符

      ?   :    三元运算符         
  变量 = (条件) ? 值1:值2
  voteable = (age < 18) ? "太年轻而不能":"年龄够";

  delete：用于删除对象中属性的   如：delete o.name; //删除o对象中的name属性
  void：   void 运算符对任何值返回 undefined。没有返回值的函数真正返回的都是 undefined。
  var iNum1=1, iNum2=2, iNum3=3; // 逗号运算符 用逗号运算符可以在一条语句中执行多个运算。
